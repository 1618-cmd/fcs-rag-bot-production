## What is This Guide

This guide explains how Admins can use the RAG Bot to get help with user management, permissions, system configuration, troubleshooting, and other administrative tasks in Vena.

---

## What is an Admin

Admins manage users, permissions, folders, processes, group permissions, and other system-level configurations in Vena. They ensure the system is properly configured and accessible to the right users.

## Tips for Best Results

1. **Be Specific About Users or Groups:** Include user names, role names, or group names when relevant

2. **Describe Permissions Needed:** When asking about permissions, describe what access is required

3. **Mention Platform:** Specify whether you are configuring Vena Desktop or Vena 365

4. **Include Error Messages:** If troubleshooting, include any error messages you see

5. **Ask for Step-by-Step Instructions:** Request detailed steps for complex configuration tasks

## When the Bot Cannot Help

If the bot cannot answer your question:

1. The question may be about client-specific configurations or custom permissions not documented in the knowledge base
2. The issue may require access to your specific Vena instance or administrative console to diagnose
3. The problem may be related to system-level issues requiring vendor support
4. The question may be about features or configurations not yet covered in the knowledge base

**What to do:**
- Review the source citations provided by the bot for related information
- Create a Jira ticket with detailed information about your configuration or issue
- Consult Vena's official documentation or support resources
- Contact Vena support for system-level or security-related issues
