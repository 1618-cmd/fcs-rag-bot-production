## What is the RAG Bot

The Vena RAG Bot is an AI assistant that answers technical questions about Vena using our knowledge base. It is available 24/7 and can help you:

- Answer questions about Vena concepts and features
- Explain VenaQL code and troubleshoot scripts
- Guide you through configuration steps
- Provide step-by-step instructions for common tasks

**Note:** The bot only answers questions based on content in our knowledge base. If it does not know something, it will tell you and help you create a Jira ticket.

## Tips for Best Results

1. **Be Specific:** Instead of "How do I use Vena?", try "How do I complete an Input Task in Vena?"

2. **Ask Follow-ups:** If an answer is not clear, ask the bot to explain further or give an example.

3. **Check Sources:** The bot provides source citations. Click them to learn more about the topic.

4. **Experiment:** Try different ways of asking the same question to get the best answer.

---

## When the Bot Cannot Help

If the bot says "I don't know" or "The context doesn't contain this information":

1. **Check Sources:** Review the source citations to see if you can find related information
2. **Create a Jira Ticket:** Click the "Create Jira Ticket" button (if available) to get support from the team
3. **Ask a Colleague:** For urgent questions or topics not yet in the knowledge base
