## Interactions

### Preview Modal
When "Preview" is clicked:
- Modal opens with document content
- Markdown rendering
- Scrollable content
- Close button

### Approve Action
When "Approve" is clicked:
- Confirmation dialog: "Approve this document?"
- Document moves: staging/ â†’ approved/
- Card disappears from Pending
- Appears in Approved tab
- Success notification

### Reject Action
When "Reject" is clicked:
- Dialog opens: "Rejection Reason" (optional text field)
- Document moves: staging/ â†’ archive/
- Card disappears from Pending
- Appears in Rejected tab
- Success notification

## Responsive Design

### Desktop (Full Width)
- 3 columns for document cards
- Full path visible
- All buttons visible

### Tablet
- 2 columns for document cards
- Truncated paths
- All buttons visible

### Mobile
- 1 column for document cards
- Stacked buttons
- Collapsible sections
