Suggested solution
Double-check the members mentioned in your prompt and try submitting it again. If you are an
AI Model Manager, try selecting ‘Refresh Model Data’ in the AI Builder, or contact Vena Support
for further assistance.
Cause
Non-Existent Name.
This error occurs when Vena Copilot attempts to map a report but cannot find a member with
the specified name. This could occur if there were data model changes where a user has
modified dimension or member names and has not yet refreshed their AI model.
