In addition to enabling Vena Copilot in your Vena environment, your Vena Admin must also
assign you the Vena Copilot role by following these steps:
1. Navigate to the Admin tab.
2. Select the Users page.
3. Select the user role(s) of the user you want to assign Vena Copilot to.
4. Select the checkbox next to Vena Copilot.
5. Select the checkmark icon to save your changes.
