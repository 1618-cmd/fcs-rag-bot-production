Error Code 1012 – Data Model Is
Out of SyncSuggested solution
Reach out to your AI Model Manager to do the following:
1. Navigate to the Modeler tab.
2. Select Data Transformations from the sidebar.
3. Select AI Builder from the sidebar tab.
4. Select Refresh Model Data.
Cause
This may occur if the data model that is linked to your AI Model has been deleted or cannot be
refreshed.
