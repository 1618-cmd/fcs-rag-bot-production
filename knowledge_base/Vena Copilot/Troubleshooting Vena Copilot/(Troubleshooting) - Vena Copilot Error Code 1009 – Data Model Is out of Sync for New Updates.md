Suggested solution
Your AI Model Manager should refresh the data model by following the steps below:
1. Navigate to the Modeler tab.
2. Select AI Builder from the left pane.
3. Select Refresh Model Data.
4. Once completed, try interacting with the AI Model again.
Cause
This issue may arise if the Reporting Agent rules are missing from the model. To resolve this,
perform a 'Refresh Model Data' to add the missing rules.
