Suggested solution 1: Ensure you have appropriate Data
Permissions
1. Select Download Report.
2. Provide the report alongside the prompt to your AI Model Manager or Vena Admin to ensure
you have the appropriate Data Permissions for the slice of data you are querying.
3. If you do not have the appropriate Data Permission, please advise your Vena Admin to
update it accordingly. Learn more about creating and managing permissions.
Suggested solution 2: Ensure you have Application
Permission to the underlying data model
Reach out to your AI Model Manager to do the following:
1. Navigate to the Admin tab.
2. Select Permissions.
3. Find the appropriate user group.
