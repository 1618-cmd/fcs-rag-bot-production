Why use this feature?
Manage all of your AI models and conversations from the AI Model page. This is where you can
edit, publish and delete AI models. For each individual AI model, you can view your users’
prompts, Vena Copilot’s responses and user feedback.
Before you begin
To follow the steps in this guide, you will need at least Modeler access and the Vena Copilot
license.
Before managing AI models, you will need to first create them. Visit the Creating an AI Model
article to learn how to do this.

Publishing your AI model makes it accessible to users. Before publishing your AI model, ensure
that you have identified the Scopes and Assumed Members.
1. Follow Manage AI models steps 1-3 to get to the AI Models page.
2. Right-click on the row or select the vertical ellipsis for the AI model that you want to
publish.
3. Select Publish from the drop-down menu.
4. You will be prompted to confirm you want to publish the AI model. Select Publish AI Model
to continue.
5. You will be prompted to assign user permissions to publish the AI model. There are two
permission types that you can assign to user groups for the AI model.
Model Management Access: Users with this access can view and edit the AI model and
conversations.
Chat UI Only Access: Users with this access can submit queries to this AI model in the
chat.
6. Select the Model Management Access and Chat UI Only Access from the drop-down
menus

Unpublish AI models
Unpublishing this AI model will prevent users from being able to launch the topic from the Vena
Copilot chat.
1. Follow Manage AI models steps 1-3 to get to the AI Models page.
2. Right-click on the row or select the vertical ellipsis for the AI model you want to unpublish.
3. Select Unpublish AI Model from the drop-down menu.
4. You will be prompted to confirm you want to unpublish the AI model. Select Unpublish AI
Model to continue.
In addition to AI model permissions, users must also have at least READ

Delete AI models
Deleting your AI model will remove it from existing AI models.
1. Follow Manage AI Models steps 1-3 to get to the AI Models page.
2. Right-click on the row or select the vertical ellipsis for the AI model you want to delete.
3. Select Delete from the drop-down menu.
4. You will be prompted to confirm you want to delete the AI model. Select Delete AI Model to
continue.
5. The AI model is now deleted.

conversations to gain insight into what questions your users are asking and how the chatbot is
responding. You can use these conversations to evaluate chatbot response accuracy and set up
Prompts for chatbot interactions.

Glossary
Builder: Located in the Vena Copilot tab. Modelers with Edit/Delete application permissions
for AI Models will be able to create, edit and refine AI models.
Published/Unpublished: An AI model is published when users can see the Vena Copilot
chatbot and ask questions. Users with a Vena Copilot user license and either Read, Update or
Delete access to the model can use the chatbot. When an AI model is unpublished, users can
no longer access it from Vena Copilot.
Steps: The number of prompts and responses generated by both the user and the AI model.
Notes & limitations
AI Models can become out of sync with data models. In this case, your users will see warnings on
the chat interface. If this occurs, select Refresh Model Data . This refreshes all AI models on your
tenant that you have access to. W e are working to automate this process in the future.
If the AI Model Manager has any data permission restrictions on the underlying Data Model for
the AI Model, they will not be able to access the Threads view containing past chats users have
had with the AI Model.
