Suggested solution
1. Reach out to your Vena Manager to ensure the process is not currently paused.
2. To check if a process is currently paused, navigate to the Manager tab.
3. Select the appropriate folder.
4. Select the appropriate Process.
5. Once in the process designer view, check the status at the top right of your screen.
6. Confirm if the process is currently paused as shown below.
7. Start the Process.
8. The contributor should now be able to successfully save data in the input template.
