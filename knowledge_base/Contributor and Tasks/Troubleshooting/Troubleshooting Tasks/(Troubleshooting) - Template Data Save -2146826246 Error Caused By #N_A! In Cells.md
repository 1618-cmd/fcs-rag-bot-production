Suggested solution
The issue is caused by a #N/A! in a data validation message cell.
1. Navigate to the Data Validation message cell on the template.
Learn to identify the Data Validation rule or message cell in Vena Desktop.
Learn to identify the Data Validation rule or message cell in Vena 365.
2. Check if there's a #N/A! error.
