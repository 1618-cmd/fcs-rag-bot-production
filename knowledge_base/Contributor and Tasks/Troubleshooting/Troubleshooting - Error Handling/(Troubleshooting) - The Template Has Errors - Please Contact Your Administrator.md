Contact your Vena Admin to complete the following steps:
1. Open the template via the Manager tab.
2. Select Analyze Template. This will show any errors on the template.
3. Fix all the template errors.
4. Save the template.
5. Once the errors have been fixed, you should be able to open the template from the
Contributor tab without any issues.
Cause
This could happen if there are mapping errors on the template.
Note
Most templates can still function with warnings present but all errors on a template
must be addressed and fixed for the template to function properly. Address all
