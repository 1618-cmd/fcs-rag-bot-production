Cause
This may happen if the MDRs are not configured with suppression ON i.e. to hide the blank rows
generated by the MDR table. With no suppression, the MDR table will pull all the possible
member combinations resulting from the MDR expression thus possibly resulting in an overflow
of the Excel Grid. Learn more about creating tables with multiple dynamic row mappings.
