Suggested solution
Contact your Vena Admin or Manager to follow these steps:
1. Log in to vena.io.
2. Navigate to the Manager tab.
3. Select the appropriate process.
4. Select the appropriate task.
5. Remove the template from the Task Forms.
6. Select Attach Forms to re-attach the template to the Task Forms.
7. Advise the Contributor to refresh their webpage and try again.
Cause
This could happen if the template property is changed from Standalone/Central to Hybrid and
