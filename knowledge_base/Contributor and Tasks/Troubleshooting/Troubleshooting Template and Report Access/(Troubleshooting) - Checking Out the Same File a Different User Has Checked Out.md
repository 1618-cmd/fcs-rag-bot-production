Suggested solution
1. If the message shows the other user's name, contact them and ask them to check in the file.
2. Contact your Vena Admin to explore the Concurrent Contributor option. Your Vena Admin
can follow these steps to turn on the Concurrent Contributor feature.  After Concurrent
Contributor is enabled, follow the steps in follow these steps to check out a file with
Concurrent Checkout. .
3. Your Vena Admin can also force check-in the file for you by following these steps.
