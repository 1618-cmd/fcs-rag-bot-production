How to
Use Inspect tool (Windows)
1. Open the template in Vena 365.
2. Right-click an empty space in the Vena 365 task pane, and then select Inspect.
3. This opens another window.
4. Select the Console tab and the Network tab to get more details.
5. If the consoles are blank, leave the window open and select Refresh.
