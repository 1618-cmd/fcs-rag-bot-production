Task Appears as Coming SoonThe user is not able to see or select the appropriate dimensions in the Choose Box. For example,
a user is looking to select an entity but there is no option to select "Entity" in the Choose Box.
Suggested solution
The following steps must be completed in the Manager tab:
1. Ensure that the Task Binding has been configured.
2. Ensure that the user has been attached to the appropriate task.
3. If there are multiple templates with similar names, ensure that the appropriate template has
