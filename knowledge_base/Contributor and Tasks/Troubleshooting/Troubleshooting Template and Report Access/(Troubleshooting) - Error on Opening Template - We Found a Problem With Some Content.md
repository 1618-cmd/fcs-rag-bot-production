Suggested solution
To resolve this issue, make a copy of the original template and apply the  following on the copy
template first. If successful, repeat on the original template. If not successful please reach out to V ena
Support:
1. Open the template.
2. Select Yes to the 'W e found a problem with some content in...' error message window .
3. Select Close  to the 'Repairs to' window that will appear .
4. The V ena template should load successfully . Select Save T emplate.
5. Close the template and re-open the template  to confirm the issue is resolved.
6. Once confirmed, replicate this on the main template.
