Need To Be Enabled andSuggested solution
1. Ensure you are using the latest version of the Vena Desktop. To confirm the latest version of
Vena Desktop, visit this article .
2. Close and then re-download the template.
3. Clear your browser's cache and re-download the template.
4. Open Vena from another web browser and download the template.
5. Try opening a different template from the one you initially had trouble opening.  If the
separate template opens successfully, try re-opening the template you were initially having
issues with.
