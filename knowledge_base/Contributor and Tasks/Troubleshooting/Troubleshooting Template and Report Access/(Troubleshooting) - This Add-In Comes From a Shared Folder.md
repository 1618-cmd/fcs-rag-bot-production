Suggested solution
1. Log in to vena.io.
2. From the top-right, select your account name.
3. Select Sideload V365 Add-In.
4. Toggle Enable Sideload to the ON position.
5. Select Save Sideload Settings.
6. Re-download and re-open the template.
Cause
This may occur if a member of the Vena team is troubleshooting an issue and has enabled the
Sideload setting.
