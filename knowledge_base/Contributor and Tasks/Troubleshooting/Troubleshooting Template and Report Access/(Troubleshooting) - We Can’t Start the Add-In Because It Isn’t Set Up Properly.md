Suggested solution
1. Please reach out to your IT team and have them follow the suggested steps in this
article from Microsoft.
2. Once completed, re-download and re-open the template.
3. If the issue persists, disable the property injectCentrallyDeployedPowerPointAddin from the
Server Properties if it is currently enabled.
4. Once completed, re-download and re-open the template.
Cause
This may occur if there's an incorrect information in the Windows registry.
