Suggested solution
1. Ensure LIDs have been enabled on the template.
2. Ensure LID buttons have not been configured to be hidden by your Vena Admin via the
Customize Ribbon window. If the Contributor Hidden box is checked next to LIDs, then it will
be hidden for Contributors. Admins can uncheck these boxes to ensure they’re visible to02/01/2026, 12:20 Troubleshooting: Vena Connector Is Missing Line-Item Details Buttons – Vena Solutions

3. Ensure the template is not opened in View-Only mode.
Cause
This may happen if the buttons have been configured to be hidden or if LIDs have not been
configured on the template.
