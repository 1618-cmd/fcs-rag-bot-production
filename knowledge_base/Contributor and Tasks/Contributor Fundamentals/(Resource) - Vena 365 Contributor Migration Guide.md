Product UpdatesOn this page, you'll find theVena 365 Contributor Migration Guide(current as of October
2023). The guide is intended for Vena users in the Contributor role that are migrating from V ena
Desktop to V ena 365, and who need to learn:
How Vena 365 is different from Vena Desktop?
Where features can be found in Vena 365?
This guide is designed for Contributor users with a Microsoft 365 subscription on Mac or
Windows.
