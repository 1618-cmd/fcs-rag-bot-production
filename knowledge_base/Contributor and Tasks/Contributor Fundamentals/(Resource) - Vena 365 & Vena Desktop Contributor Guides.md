On this page, you'll find the Vena 365 Contributor Guide (current as of June 2022), as
well as the Vena 365 Contributor Guide for Tasks (current as of July 2025 for the new
Tasks experience). The guide is intended for Vena Cloud users who hold the Contributor
role, and who need to learn how to:
Access and use the Vena web app at vena.io
View (download) reports
Work on input and review tasks
Use Vena 365 for Excel
This guide is designed for Contributor users with a Microsoft 365 subscription on Mac or
Windows.
Visit this page for the Vena Desktop Contributor Guide.

How to use this resource

The Vena 365 Contributor Guide is available in both PDF and Word format.
We recommend using PDF format as the guide is intended to be viewed on a digital
device.
While you can print out a physical copy if desired, we recommend using the digital PDF as
the guide contains bookmarks, weblinks, etc. that are only available in digital format.
To download the Vena 365 Contributor Guide (in either PDF or Word format) or the
Vena365 Contributor Guide for Tasks, select the appropriate file under the Attachments
section at the bottom of this page.
