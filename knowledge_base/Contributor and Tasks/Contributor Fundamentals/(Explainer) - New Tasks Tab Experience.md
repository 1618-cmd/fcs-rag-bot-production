Tasks tab terminology
Here are terms you should be familiar with while using Tasks tab.
Icons:
Ready for Input: Shown beside a form on an input task when the file is not in use by other
users and is available and ready for input (for non-Concurrent Contributor only). This is
always shown when Concurrent Contributor is enabled on the task.
Ready for Review: Shown beside a form on a Review task when the file is checked in and is
available and ready for review (for non-Concurrent Contributor only). This is always shown
when Concurrent Contributor is enabled on the task. Also, this is only shown when Allow Data
Saves is enabled on the Review Task.
No Edit Access: Shown beside Review task forms where Allow Data Saves is not enabled.
Non-Excel File: Shown beside any form and task when the form is not an Excel file.02/01/2026, 11:12 Explainer: New Tasks Tab Experience – Vena Solutions
https://support.venasolutions.com/hc/en-us/articles/36001963275277-Explainer-New-Tasks-Tab-Experience 12/14

Watcher: Shown beside any form belonging to a task where the user is a Watcher and not an
owner/support worker.
Call to Action (CTA) Buttons:
Input: Download a file from an Input task and input your data.
Review: Download a file from a Review task and review the data.
Return File: Make a file available for other users from any task.
View: File is either a form on a Review task or is not an Excel file.
