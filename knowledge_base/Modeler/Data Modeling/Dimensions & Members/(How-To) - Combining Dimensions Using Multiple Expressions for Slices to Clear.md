There may be times when you need to combine dimensions to define a more specific slice of
data to be cleared. Using multiple expressions can achieve this.
Before you begin
To follow the instructions in this article, you will need at least Modeler access.
How to

2. Select Data Transformations.
3. In Channels, select the appropriate Destination.

combine dimensions to form a more specific slice of data.
