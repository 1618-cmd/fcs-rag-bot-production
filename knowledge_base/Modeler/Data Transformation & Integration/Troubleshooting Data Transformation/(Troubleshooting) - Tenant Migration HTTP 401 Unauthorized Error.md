Troubleshooting: Vena Import
& Export API 401 Unauthorized
Status Code1. Ensure you have entered the correct complete application token.
2. Ensure the domain you are entering is for the destination tenant and not the source tenant.
3. Ensure the token being used exists in the destination tenant.
4. Ensure there is no blank space when copying and pasting the application token.
5. If the issue persists, refresh the webpage and try again, or create a new application token in
the destination tenant.
Cause
This may occur if the application token entered is incorrect or if you are entering an incorrect
destination domain.
