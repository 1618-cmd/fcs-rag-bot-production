Suggested solution
1. This error is encountered when a parent folder of a migrated Pre-Configured Solution
Package resource is deleted. As a best practice, always delete a process or an integration
component by selecting that specific item. Avoid deleting the parent folder itself.
2. Please reach out to the Support Team.
