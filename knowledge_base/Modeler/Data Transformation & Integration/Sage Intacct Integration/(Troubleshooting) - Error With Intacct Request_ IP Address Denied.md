Suggested solution
IP filtering may be enabled on the Sage Intacct instance. In the Sage Intacct instance, to resolve
this:
1. Navigate to Company.
2. Select Security.
3. Select Password.
4. Select IP Filter.
5. Review the filtering to confirm if this is required and update accordingly.
6. Once updated, try setting up the data feed again.
Cause
This can happen if IP filtering is enabled on the Sage Intacct instance.
