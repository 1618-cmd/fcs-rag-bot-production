Suggested solution
1. Check with your CSM to confirm that you have this feature enabled in your Vena
environment.
2. Once the feature has been enabled, please follow the steps in this article to clear the current
permission in your Vena Insights Connector application.
3. Create a new application token in Vena and add the necessary data permissions as required.
4. Go back to your Vena Insights Connector application.
5. Select Get Data and follow the process to connect to your Vena data models. Ensure you are
using the newly created token.
6. If the issue persists, ensure you have upgraded your Vena Insights Connector application to
the latest version.
7. If there's still an issue, uninstall the Vena Insights Connector application and install it directly
from the Power BI website instead of the one from the Microsoft Windows Store.
8. If uninstalling and reinstalling do not work, please check that this setting is enabled.
