– External Sources – Data FeedsThe Vena Map by Name feature allows you to streamline your channel mappings by
automatically matching Sources to Destinations with the same name. This functionality enables
you to save time, particularly for channels that require a large number of mappings. We have
also added a button that allows you to delete all of your connections instantly as well.
Before you begin
 To follow the instructions in this article, you will need at least Modeler access.
