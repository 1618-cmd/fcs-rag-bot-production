Power BI Integration
Sage Intacct IntegrationSuggested solution
1. Ensure you have an ETL template with that ID in your data model.
2. Ensure the application token/credential used for the Power Automate flow has all the
necessary application permissions.
3. If the ETL template ID exists but the ID is different, update it in Power Automate.
4. If there's no ETL template with that ID or it was recently deleted, create a new ETL template
as required and update the ID in Power Automate with the template ID.
Cause
This may happen if Power Automate tries to run or trigger an ETL job but the system cannot find
the specified ETL template ID.
