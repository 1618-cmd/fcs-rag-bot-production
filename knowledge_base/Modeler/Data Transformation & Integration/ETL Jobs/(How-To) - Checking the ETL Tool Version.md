Why use this feature?
Do you want to find out what version of the Vena ETL tool you are using? You can easily check
the tool version using your Command Prompt on your computer. If there is ever an issue with
the ETL tool, you will need to ensure that you are using the most recent version.

Open Command Prompt on your desktop. Make sure that you are in the same folder where the
ETL JAR file is located. These steps apply to Windows users only.
How to
1. Run the following in the command line:
java -jar etl.jar --version
2. Once you run the above command, you will see the version in the second line.
