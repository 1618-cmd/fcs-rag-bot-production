Vena's Fall '25 release is here! Explore the latest updates and enhancements available to further
empower your office of finance.
About the release
Vena's Fall 2025 release advances AI-driven data insights for our customers by introducing
intelligent agentic-first, natural language query generation and enhanced visualization tools that
transform how finance teams work.
Feature spotlight
New features include:
Vena Copilot’s Reporting Agent
Updated Tasks tab for Contributors
Dynamics 365 F&O Integration
Vena’s Query Agent for MQL
In this overview, learn more about each feature included with this release, available
starting November 12 2025.

Release Overview
Ad Hoc deprecation. With Vena’s new Ad Hoc tool now generally available, the legacy Ad Hoc
tool will be deprecated and removed from the platform on December 15, 2025. Learn more
about the new Vena Ad Hoc tool.
Role-based licensing. Role-based user licensing assigns license types according to user roles,
offering enhanced visibility into license usage. This approach ensures greater transparency
and simplifies the monitoring of contract compliance. Learn more about role-based
licensing.
