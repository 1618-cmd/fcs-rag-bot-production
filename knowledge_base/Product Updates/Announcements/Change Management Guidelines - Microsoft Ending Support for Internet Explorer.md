What's New in Vena: Fall '23On June 15, 2022, Microsoft is ending support for Internet Explorer 11. Following this change,
Vena will no longer support Internet Explorer for users accessing the Vena platform (vena.io).
For more information on this change, check out Microsoft's announcement on ending support
for Internet Explorer in this article.
Q & A
What is the change?
On June 15, 2022, Microsoft will end support for Internet Explorer 11 browser. As a result of this
change, beginning on June 15, 2022, Vena will no longer:
Guarantee proper functioning of the Vena application running in Internet Explorer.
Provide support for any issues users may encounter in the application when using Internet
Explorer.
Test new versions of the Vena application in Internet Explorer.
Why is this happening?
The decision to end support for Internet Explorer is part of Microsoft's strategic growth plan that
