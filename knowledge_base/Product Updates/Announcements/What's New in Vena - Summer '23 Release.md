Release overview
All of the other exciting features included with this release
Vena 365 Connect - Copy and Paste: Save time building templates and reporting by reusing
existing mappings via Vena Copy and Paste. Learn more here.
Tasks - Tasks tab is now available:  A new, cleaner and easy-to-use Contributor experience
is waiting for you. Starting now, all customers can benefit from the new and improved
interface. Learn more here.
For more learning opportunities, check out the following Vena Academy courses
related to the exciting features included in this release:
Tasks in Vena 365 Connect: This 10 minute course focuses on Tasks in V365C.
Tasks in Vena Desktop: This 10 minute course focuses on Tasks in Vena Desktop.
Working with Contributors in Vena 365: This hour-long course includes Tasks as well as
Contributor functions in Excel (LIDs, Insert Row, Drill Tools, etc.)
Working with Contributors in V ena: This hour-long course shows how V ena makes it possible for
you and your Contributors to provide inputs, collaborate, and gain insights through features such
as tasks, comments, data inspection tools and more.
