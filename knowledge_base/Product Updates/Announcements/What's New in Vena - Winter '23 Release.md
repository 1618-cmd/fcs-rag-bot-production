Vena 365 Connect password protect
Password protect your templates in Vena 365 Connect. Use Vena 365 Connect password protect
to prevent unwanted input or formula changes. You can choose either Vena’s default password
or set a custom one. Protect your templates and share them with ease, without having to worry
about unwanted changes. Read the How To: Protecting a Vena 365 or Desktop Template
Worksheet with a Custom Password for more information about password protection.
