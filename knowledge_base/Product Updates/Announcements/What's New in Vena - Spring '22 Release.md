rules as Hard or Soft Validations and include customizable messages to Contributors for your

More Granular and Detailed Planning
Vena 365 Connect Staging Queries
Use Vena 365 Connect Staging Queries for even more detailed planning abilities! Staging
Queries allow you to retrieve data from any data table created from the ETL import process and
apply it directly in your template. Choose which columns you want included and use queries to
return only the data that you want to see. For more information about Vena 365 Connect

Customizable Templates and Views
Vena 365 Connect Expand/Collapse
Expand/Collapse for Vena 365 Connect is here! Use Expand/Collapse with multi-dynamic rows
to view your data from different levels. Use Expand to view descendants, children and bottom-
level members or use Collapse to only see parent members. With Expand/Collapse, you can
prepare detailed reports tailored to your audience and investigate variance with a single click.

Simpler and More Intuitive
Workforce planning pre-configured solutions
The workforce planning pre-configured solutions have many performance improvements for
a simpler, more intuitive experience. The Jump To… feature allows for easier navigation as
hyperlinks jump to different areas of the template. Drivers/Thresholds is now next to their
associated benefit groups; adding and removing benefits from the template is all in one location,
allowing for smoother formula referencing. Benefit-Specific Formulas now make updating,
customizing and understanding formulas even easier. For more information about workforce
planning pre-configured solutions, check out this page.
