New/Old Modeler Button: Easily switch between the new and old modeler user interface
with the click of a button.
Try the new UI: Users that have both Modeler and Admin rights can find the “Try The
New Modeler UI” button in the green bar at the top of the screen (next to the Vena Help
Center).
Revert back to old UI: Users that have both Modeler and Admin rights can switch back to
the old UI by navigating to the Modeler tab and clicking the “What’s New” button found in
the bottom-left corner of the screen. The What’s New pop-up will then open, which
contains the “Go Back To Old UI” button.
