March 25, 2024Bug fixes
This fixed a bug where an error would occur when a user checked in a Review file. Users will
no longer see errors that use forms from previous tasks.
This fixed a bug where Excel crashed when a user attempted to delete certain intersection
comments.
New features and enhancements
Previously, page breaks were not carried over when Cascade was performed. Now, all sheets
created by Cascade will have page breaks copied from the original sheet.
New templates will have the setting TextFormatForDrillTransactions set to True by default.
Learn more about the TextFormatForDrillTransactions setting.
