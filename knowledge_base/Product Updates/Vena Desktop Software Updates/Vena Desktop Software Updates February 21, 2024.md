Note: Code Signing Certificate Update for Vena Desktop
In this release we updated the security certificate for the Vena Desktop Add-in. As a
result, you may see a system prompt during installation asking you to verify that you
trust the publisher – this is expected.
In some cases, strict security settings may cause the Vena Desktop Add-in to close
unexpectedly. If this happens, work with your IT team to add a security exception to
re-confirm the Vena Desktop Add-in is trusted.
