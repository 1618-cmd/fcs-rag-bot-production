March 25, 2024New features and enhancements
The user interface for the Customize Refresh screen has been updated to improve clarity
and reduce disk space usage during Vena installations.
The Vena Desktop installer now verifies that .NET Framework 4.6.2 is installed. This version is
included by default in Windows 10 Anniversary Update (Version 1607) and later, including all
versions of Windows 11 and Windows Server 2016 and above.
