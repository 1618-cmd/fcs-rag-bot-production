March 25, 2024Bug fixes
Opening templates with Line-Item Details through Vena Desktop and Vena 365 is now a more
streamlined experience. Users no longer see a warning message when refreshing templates
in Vena Desktop.
If a session times out, users now see a prompt to download the template again to renew the
session. This replaces a previous in-app browser experience that no longer worked due to
Microsoft's retirement of Internet Explorer.
