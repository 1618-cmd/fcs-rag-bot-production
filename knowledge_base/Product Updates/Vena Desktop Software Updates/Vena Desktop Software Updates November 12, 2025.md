March 25, 2024New features and enhancements
Improved error messages now provide clearer guidance to help you resolve template issues.
Vena Desktop now uses a unique user-agent string (“VenaDesktop/[version]”) to better
identify legitimate network requests. This update helps prevent issues with security software
that may have previously blocked Vena Desktop traffic.
Added a new pop-up to warn users when the session token is set to expire in 1-2 minutes.
