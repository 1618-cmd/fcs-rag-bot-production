Vena Desktop Software Updates
March 25, 2024New features and enhancements
Users no longer need to opt in to use the Multi-Data Model functionality in Vena Desktop.
For security purposes, when the File State Management response is logged to vena.web.log,
the API properties are now omitted.
For security purposes, authorization headers have been removed from vena.web.log.
Users now receive a warning message if they try saving data after a failed refresh to prevent
saving stale data.
