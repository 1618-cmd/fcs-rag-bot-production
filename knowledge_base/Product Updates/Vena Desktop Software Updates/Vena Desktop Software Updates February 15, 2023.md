Bug fixes
Fixed a bug that caused hide flags to take precedence over show flags in a template. If a row
has both #showrow and #hiderow or a column has both #showcolumn and #hidecolumn,
now the show flags will take precedence over the hide flags.
Fixed a bug where the view function in Drill Saves was not highlighting the correct
intersection after the user performed Drill Down. Now, the correct intersection is highlighted
and is opened in the correct sheet when the user selects View in Drill Saves.
