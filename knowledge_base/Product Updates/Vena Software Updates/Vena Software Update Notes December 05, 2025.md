Product UpdatesVena Software Update Notes
December 05, 2025
New features and enhancements
Ad Hoc: A new Suppress Zero feature is now generally available within Vena Ad Hoc. Learn
more about Suppress Zeros and how to automatically exclude null ("0") values when your
sheet refreshes.
Admin: A new button ("Reset 2FA") allows administrators to control 2FA at the user level.
Administrators can reset or remove 2FA for a user as appropriate.
Admin: We've updated the Application Token page with a clean, modern user interface that
makes it easy for you to create and use tokens on our platform.

Bug fixes

Existing Zeros", the Line-Item-Details attached to the intersection may be deleted.
