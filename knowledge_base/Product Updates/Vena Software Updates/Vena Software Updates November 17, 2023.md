Bug fixes
Vena 365: This change fixed a bug preventing templates with LIDs from subtotaling correctly.
Previously, when a template with multiple row ranges (connected via Wildcards and with LIDs
enabled) was refreshed, the LID subtotal formulas would go to unintended rows. This bug is
now resolved, and subtotal formulas on templates will work correctly.
Vena 365: This fixed a bug that was showing the default dimension member instead of the
Alias for Alias only members. With this fix, the tables now have the correct member
