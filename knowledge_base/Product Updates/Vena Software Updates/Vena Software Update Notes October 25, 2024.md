Product UpdatesVena Software Update Notes October
25, 2024

New features and enhancements
Vena for Microsoft PowerPoint: Templates will now be ordered alphabetically.
Vena for Microsoft PowerPoint: Visuals will now appear in the order they're in in Excel
rather than in the order visuals were added to PowerPoint.
Vena for Microsoft PowerPoint: Users will no longer see "pptdash" in front of visuals loaded
into the PowerPoint library.
