Bug fixes
Manager V365C: This fixed a bug that was causing issues with task bindings. Users that had
permissions for a process but not explicit file permissions could not add task bindings and
would receive an error. Now, users can add task bindings successfully.
Modeler: This fixed a bug that was preventing users from successfully deleting cloned data
models. Now, users that were previously unable to delete a cloned model are able to do so.
