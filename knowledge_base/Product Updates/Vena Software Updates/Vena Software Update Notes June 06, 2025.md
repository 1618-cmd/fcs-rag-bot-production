New features and enhancements
Modeler: From now on, Standard Member auto-assignment for the Period dimension will
use the following rules:
Months (Jan-Dec) will be assigned based on an alias.
The member name of Months must be an integer to be auto-assigned.
All other members of the Month dimension will be auto-assigned based on name, not
alias.
