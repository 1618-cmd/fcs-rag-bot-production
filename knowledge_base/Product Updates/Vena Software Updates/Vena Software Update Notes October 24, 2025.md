New features and enhancements
Modelers: Those with NetSuite integrations must download the new Vena (1) - Saved
Searches and Roles (Bundle ID 315383) to version 1.11 for improved security and ease of use.
Note that due to these updates, Consumer Key and Consumer Secret are no longer required.
Managers: When attempting to reset a paused process, a warning message will pop up with
the option to cancel or reset the process.
