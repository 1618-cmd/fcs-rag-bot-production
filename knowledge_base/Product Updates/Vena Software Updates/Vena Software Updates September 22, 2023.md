New features and enhancements
Manager:An enhancement has been made to Static Row and Static Column Mappings. Now
a dimension must be selected before you are able to use Automap. This guarantees that
when Automap is used, the members from only that dimension are retrieved and non-
ambiguous mapping is avoided.
