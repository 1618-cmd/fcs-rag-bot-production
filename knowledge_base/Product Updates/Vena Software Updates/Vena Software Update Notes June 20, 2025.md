Product UpdatesVena Software Update Notes June 20,
2025
New features and enhancements
Vena PowerPoint: Users can now create a blank PowerPoint file from the Global Files
Library.
Vena Copilot: For "Create AI Model" Application Permissions, the customer will now see only
the "All" option for the AI hierarchy if they have access to the resource group.
Calc Scripts: Users can now unhide up to 13 DAX measures in the Scripts tab to build new
measures.

Vena Software Update Notes May
23, 2025Bug fixes
ETL: Fixed a bug where new data saves and ETL imports were failing after a user cancelled an
"Add Dimension" or "Delete Dimension" job.
Insights: Fixed a bug that displayed Power BI permissions in the Application Permissions
drawer. This fix filters out Power BI permissions in the Application Permissions drawer so
they’re not deleted by mistake. Power BI permissions may be managed via the Insights tab.
Template Studio: Fixed a bug where the Vena Desktop ribbon wouldn't load when opening
some Template Studio files.
