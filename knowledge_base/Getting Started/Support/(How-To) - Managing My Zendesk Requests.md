To follow the instructions described in this article, you must have a Vena account.
How to
Option 1: From your Help Center profile
1. Log in to vena.io.
2. Select Help in the top-right corner. This launches Vena Help.
3. Select Knowledge Base.
4. Select your username in the top-right corner.
5. Select Requests.
