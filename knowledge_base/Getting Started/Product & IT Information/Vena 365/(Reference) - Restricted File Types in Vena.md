Reference guide
Restricted file types
The following file types can potentially be risky to download and Vena will therefore prevent
users from uploading them. While the exact list may change over time, the general guidance is
that any file that can be run as a program (executable, script, installation file) will be blocked
from uploading.
The file types are:
APPLICATION
BAT
EXE
JAR
JS
JSON
LNK
MSIVBS
COM
CMD
ASP
JSP
PHP
Unix shells
