Option 1:Conflicting Add-ins
Conflicting Add-ins can prevent Vena templates from functioning as expected. If you have other
Add-ins installed, try disabling them to see if that resolves the issue.
1. Open Excel.
2. SelectFile.
3. SelectOptionsto open the Excel Options menu.
4. SelectAdd-insin the sidebar on the left.
5. Select ExcelAdd-ins from the Manage: menu, then select Go... This opens a pop-up window
showing all Excel Add-ins.
6. Uncheck any non-Vena Add-ins, then select OK. This closes the Excel Options window.
7. Repeat steps 2-4 to re-open the Excel Options window and navigate to the Add-ins section.
8. Select COMAdd-ins from the Manage: menu, then select Go... This opens a pop-up window
