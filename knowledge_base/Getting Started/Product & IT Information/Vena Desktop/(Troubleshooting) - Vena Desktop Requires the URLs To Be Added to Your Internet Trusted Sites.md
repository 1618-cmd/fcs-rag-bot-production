Suggested solution
1. Follow the steps outlined in the How-To article on Adding Vena to Trusted Sites.
2. If you are still getting the same warning message after following the article above, you can
select "Install Addin" anyway. This is a warning message and the system should still allow
installation to proceed, or you can also download and install Vena Desktop directly from
http://addin.vena.io/release/vena.application.
3. If there are any issues after installation, please reach out to the Support Team and provide
the add-in log file located in %appdata%\VenaSolutions\vena.log
