You can hide optional buttons, mostly in the Contributor section. Core functions like Choose or
template mapping tools cannot be hidden. Some buttons (e.g., Zoom In/Out and Cascade)
appear in both Manager and Contributor sections.
Any changes made to the ribbon only apply to the specific template where you made the
change. You can choose to hide buttons from either Managers, Contributors or both. Hidden
buttons will also apply to Mac and Office Online versions.
Before you begin
To follow the instructions in this article, you will need Manager access. In addition, you will need
to use the Windows version of the Vena Desktop (in conjunction with Excel for Windows).
