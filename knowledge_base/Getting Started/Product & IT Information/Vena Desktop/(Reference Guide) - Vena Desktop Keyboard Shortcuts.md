Reference Guide
General Usage
This guide covers Windows-only Vena-specific shortcuts for ribbon navigation. For general Excel
shortcuts, refer to the documentation provided by Microsoft.
To access Vena Desktop shortcuts, press Alt+Y+1 sequentially or simultaneously.
Once activated, Keytips appear, showing shortcut labels. These update contextually as you move
through the ribbon.
Note: Vena Desktop has two ribbons, Manager and Contributor. Shortcuts are contextual and
may overlap depending on the active ribbon.
